### Instrumen Kuesioner Pengujian Beta - Toko191

Berikut adalah daftar pertanyaan kuesioner yang digunakan untuk menguji kepuasan pengguna terhadap aplikasi Toko191. Kuesioner ini mencakup aspek Usability, Information Quality, Interaction Quality, dan Kepuasan Pengguna.

| 1 | Apakah antarmuka (interface) aplikasi Toko191 mudah dipahami dan digunakan oleh pemula? | | | | | |
| 2 | Apakah pemilihan warna dan font pada aplikasi sudah tepat dan nyaman dipandang? | | | | | |
